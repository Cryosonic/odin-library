# odin-library
## Objective
A project on using Javascript Object constructors.
This project demonstrates the ability to use functional object constructors and prototype inheritance.

## Revisted
This project was later revisted. Using the basic template that already exists, class constructors were
used in place of the functional object constructor.

## Features
An exciting part of this project was the use of nested selectors for the CSS stylesheet.
This makes the stylesheet easier to read and maintain as it follows the same flow and layout as the HTML document.
